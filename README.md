# 照片经纬度写入

- 经纬度拾取依赖高德地图，使用前请先申请高德地图API [参考](https://lbs.amap.com/api/javascript-api-v2/prerequisites)
- 无损写入经纬度至EXIF数据中
- 写入模式为追加，如果照片本身不包含EXIF数据则无法写入
